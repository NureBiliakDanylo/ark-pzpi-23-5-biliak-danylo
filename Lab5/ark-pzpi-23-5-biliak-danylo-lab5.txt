3 ХІД РОБОТИ
На основі досвіду створення та розгортання частин сервісу у минулих лабораторних роботах було створено PowerShell-скрипт deploy.ps1 для автоматизації розгортання сервера на ОС Windows.
Скрипт розгортання серверу виконує такі кроки:
1. Перевіряє наявність Node.js в системі.
2. В інтерактивному режимі запитує у користувача параметри для підключення до PostgreSQL, пропонуючи значення за замовчуванням.
3. Автоматично створює конфігураційний файл .env.
4. Встановлює всі залежності за допомогою команди npm install.
5. Виконує скрипт ініціалізації бази даних.
6. Запускає сервер.
Також було створено невеликий файл README.md з інструкціями щодо використання скрипту.
Далі було виконано створення скрипту розгортання для IoT-клієнта: Було розроблено концепцію розгортання для вбудованого пристрою, яка полягає у компіляції прошивки та її завантаженні в пам'ять мікроконтролера.
Створено PowerShell-скрипт deploy_iot.ps1, який:
1. Перевіряє наявність в системі PlatformIO Core CLI (pio).
2. Інформує користувача про необхідність підключення пристрою до комп'ютера.
3. Виконує команду pio run --target upload для автоматичної компіляції та прошивки.
Для полегшення роботи було створено окремий README.md в директорії iot-client з поясненням передумов (драйвери, PlatformIO) та процесу прошивки.

4 ВИСНОВКИ
В результаті виконання лабораторної роботи було успішно розроблено та реалізовано два автоматизовані скрипти для розгортання серверної та iot-клієнтської частин програмного комплексу. 
Створений інструментарій дозволяє значно спростити та прискорити процес налаштування системи на нових машинах з ОС Windows. Скрипт для сервера автоматизує встановлення залежностей, конфігурацію підключення до бази даних та її ініціалізацію, мінімізуючи ризик помилок через людський фактор. Інтерактивний режим запиту даних робить його гнучким для різних середовищ. Скрипт для IoT-клієнта та супровідна документація слугують мостом між розробкою в емуляторі та розгортанням на реальному обладнанні. Це надає чіткий і простий механізм для прошивки пристроїв ESP32, що є критично важливим для тестування та впровадження IoT-рішень.
